Инструкция по запуску SovaPet WebApp Bot:

1. Установи Node.js (если ещё нет): https://nodejs.org
2. Распакуй архив.
3. Открой терминал в папке проекта.
4. Установи зависимости:
   npm install
5. Запусти бота:
   npm start

После этого перейди в @sovapet_game_bot в Telegram и напиши /start — появится кнопка WebApp.
